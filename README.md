# github-config

这是一个 **GitHub 配置仓库（GitHub Configuration Repository）**  
用于记录和管理本人在 GitHub 使用过程中的配置、约定和说明。

---

## 一、仓库用途（Purpose）

本仓库 **不用于项目开发**，仅用于：

- 记录 GitHub / Git 的基础配置说明
- 保存个人使用 GitHub 的规范与习惯
- 作为长期保留的配置参考仓库

---

## 二、包含内容（Possible Contents）

- Git 使用说明（非代码）
- GitHub 相关配置说明
- 历史配置记录与备注

---

## 三、使用说明（Usage）

- 本仓库不作为项目仓库使用
- 不接受功能性代码开发
- 主要用于个人管理与长期留存

---

## 四、状态说明（Status）

- 类型：配置仓库
- 状态：长期保留 / 低频更新
