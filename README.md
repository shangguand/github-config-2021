# GitHub-config-2021

这是一个 **GitHub 配置仓库（GitHub Configuration Repository）**  
用于记录和管理本人在 GitHub 使用过程中的配置、约定和说明。

---

## 一、仓库用途（Purpose）

本仓库 **不用于项目开发**，仅用于：

- 记录 GitHub / Git 的基础配置说明
- 保存个人使用 GitHub 的规范与习惯
- 作为长期保留的配置参考仓库

---

## 二、包含内容（Possible Contents）

- Git 使用说明（非代码）
- GitHub 相关配置说明
- 历史配置记录与备注

---
---

## 三、使用原则（Rules）

- 本仓库 **不存放业务代码**
- 不作为项目开发仓库
- 所有内容以「说明 / 文档 / 记录」为主
- 修改需写清时间与原因

---

## 四、维护记录（Log）

- 2021：初始化 GitHub 配置仓库
- 2026：重新整理与规范用途说明
